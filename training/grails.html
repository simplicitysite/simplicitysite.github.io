Grails Core Training

 

This intensive 4-day Groovy & Grails course will help you and your team to quickly move beyond the basics onto the most powerful aspects of both Groovy and Grails. Through intensive, hands-on interactive lab work, you will learn many advanced aspects of both the Groovy programming language and the Grails web application framework- so bring your coding brain!

LEARN HOW TO:
  • Write programs using Groovy
  • Integrate with Java
  • Use Grails for Web development and persistence
  • Have a sound knowledge of all the core concepts from the Web to the persistence layer
  • Understand key development best practices that enable the development of higher quality code
