http://grails.org/plugin/serializable-session

Grails Serializable Session Check

 

Grails is one of the premier rapid development frameworks available today. Using the power of the de facto standard Spring software stack, combined with Groovy, Hibernate and Sitemesh, sites and complex applications can be developed with ease.

Tomcat is the default deployment container for Java web applications, and is used during the development of Grails applications

This plugin allows a development check to be enforced that ensures any session data is fully serializable, and so compatible with a stored session strategy, such as implemented by MongoDB Tomcat Session Manager

Documentation is maintained in the GitHub repository.
