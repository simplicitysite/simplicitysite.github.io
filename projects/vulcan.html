Vulcan

 

Vulcan is the framework we use to perform deployments, migrations and backups in virtualised/ cloud environments.

It allows a developer to write a specification of the deployment, along with assertions that the task has been run correctly. Together, this allows deploy specifications to be developed in a test driven way, allowing the ideal of Infrastructure as Code to be fully realised.

The Vulcan source code is currently being migrated, please check back soon for access.
