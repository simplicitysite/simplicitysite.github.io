Mongo Tomcat Session Manager

 

MongoDB is a rich document oriented and highly performant database.

Tomcat is the default deployment container for Java web application.

This component allows Tomcat sessions to be stored in MongoDB and is suitable for high performance clustering.

Documentation is maintained in the GitHub repository.
