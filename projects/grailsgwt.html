Grails GWT Plugin

 

Grails is one of the premier rapid development frameworks available today. Using the power of the de facto standard Spring software stack, combined with Groovy, Hibernate and Sitemesh, sites and complex applications can be developed with ease.

Google Web Toolkit (GWT) is a browser based rich client development framework that allows Java to be used to create rich and compelling applications that run within a browser, with no plugins.

The GWT Grails Plugin combines the GWT framework into Grails, to allow applications using both system to be easily developed, and for code to be shared between the client and server to allow flexbile validation and superior testing
